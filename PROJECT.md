# Our Bruges Project

**Project:** Personalised NFC GitHub Pages website\
**Repository:** `our-bruges`\
**Status:** Active (Version 2)

------------------------------------------------------------------------

# Vision

Create a simple, elegant website for our September 2026 Bruges trip.

Each traveller receives an NFC tag as part of their gift. Scanning the
tag opens their own personalised webpage.

The website should feel like a luxury boutique hotel welcome pack rather
than a travel itinerary.

**Principles**

-   Relaxed
-   Elegant
-   Personal
-   Curated
-   Mobile-first
-   Simple HTML/CSS
-   GitHub Pages
-   No unnecessary features
-   Avoid scope creep

# Overall Tone

This is not a checklist.

The itinerary provides gentle anchors rather than a timetable.

The emphasis is on slowing down, enjoying each other's company and
allowing Bruges to surprise us.

The pages should encourage wandering rather than rushing.

The trip is curated by Dan but should never feel over-planned or
restrictive.
